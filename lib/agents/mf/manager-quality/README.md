# Manager-quality agent

Tier of fund house, scheme age, manager tenure, alpha vs category. Weight: **25%**.

## v1 inputs
- Fund house allowlists (tier1/tier2)
- Inception date
- Manager tenure (manual / scraped)
- Trailing 3y return vs category median (alpha proxy)
- AUM in crore

## v2 ideas
- Structured fund-manager DB (career, prior fund returns, awards)
- Star ratings (Morningstar / Value Research) as a soft signal
- Manager exit alerts (when star manager leaves, score drops fast)
